= Index Category Key
:navtitle: Index
:source-language: yaml

The key in the `index` category controls the number of columns in the autogenerated index section.

[cols="4,4,6a"]
|===
|Key |Value Type |Example

|column_gap
|xref:measurement-units.adoc[Measurement] +
(default: `$base-font-size`)
|[source]
index:
  column_gap: 12

|columns
|Integer +
(default: `2`)
|[source]
index:
  columns: 2
|===
